## FAQ

### 为何基于已汉化过的项目再次进行汉化?

`SlimefunGuguProject/Slimefun4` 改动了一些东西，某些物品的行为与上游不一致。

### 改动了什么其他内容吗？

仅改动了需要汉化的部分文本。

### 稳定性如何呢？

已通过所有测试用例，在 1.20.6 测试并无问题，理论上只要是官方支持的版本都没问题。
